## 二、准备工作（第一次，10分钟）

### 2.1 注册 Vercel

1. 打开 https://vercel.com
2. 点击右上角 **Sign Up**
3. 选择 **Continue with GitHub**（推荐，自动关联代码仓库）
4. 授权 Vercel 访问你的 GitHub 账户
5. 完成注册

### 2.2 注册 Etherscan 获取 API Key（免费）

1. 打开 https://etherscan.io/register
2. 填写邮箱、用户名、密码
3. 登录后进入 https://etherscan.io/myapikey
4. 点击 **Add** 创建新 Key
5. 复制 API Key 备用

### 2.3 安装 Vercel CLI（可选，也可以用网页部署）

```bash
npm install -g vercel
```

## 三、部署步骤

### 方式一：通过 Vercel 网页部署（推荐，最简单）

1. 把项目代码推送到 GitHub 仓库
   ```bash
   cd C:\Users\LC\projects\defi-saas
   git init
   git add .
   git commit -m "DeFi Security SaaS v2.0"
   # 在 GitHub 创建新仓库，然后：
   git remote add origin https://github.com/你的用户名/defi-security-saas.git
   git push -u origin main
   ```

2. 打开 https://vercel.com/new
3. 点击 **Import** 你的 GitHub 仓库
4. 配置：
   - Framework Preset: **Other**
   - Root Directory: `. /`（默认）
   - Build Command: 留空（或填 `npm install`）
   - Output Directory: 留空
5. 展开 **Environment Variables**，添加：
   | 变量名 | 值 | 说明 |
   |--------|-----|------|
   | `ETHERSCAN_API_KEY` | `你的Etherscan Key` | Etherscan API |
   | `ADMIN_API_KEY` | `admin-随机字符串` | 管理员 Key，用于生成订阅码 |
   | `NODE_ENV` | `production` | 生产模式 |

6. 点击 **Deploy**
7. 等待 1-2 分钟，部署完成
8. Vercel 会自动分配一个域名：`https://你的项目名.vercel.app`

### 方式二：通过 Vercel CLI 部署

```bash
cd C:\Users\LC\projects\defi-saas
vercel login
vercel
# 按提示配置，首次部署成功后，后续更新只需：
vercel --prod
```

## 四、绑定自定义域名

### 4.1 买域名（约 50 元/年）

推荐平台（中国大陆友好）：
- **Namesilo**：https://www.namesilo.com （支持支付宝，$1-10/年）
- **Cloudflare**：https://www.cloudflare.com/domains （原价，但 DNS 服务好）
- **腾讯云/DNSPod**：https://dnspod.cloud.tencent.com （国内速度快）
- **阿里云万网**：https://wanwang.aliyun.com

域名建议：
- `defiscan.cc`（便宜，约30元）
- `chainshield.xyz`（有个性）
- `defiguard.pro`（专业感）
- 或直接用你的个人品牌域名

### 4.2 在 Vercel 配置域名

1. 打开 Vercel 项目仪表盘 https://vercel.com/dashboard
2. 点击你的项目 → **Settings** → **Domains**
3. 输入你的域名，如 `defiscan.cc`
4. Vercel 会给出需要配置的 DNS 记录

### 4.3 配置 DNS

去你的域名注册商后台，添加 DNS 记录：

**如果使用根域名（如 defiscan.cc）：**
```
类型: A
名称: @
值: 76.76.21.21
```

**如果使用 www 子域名（如 www.defiscan.cc）：**
```
类型: CNAME
名称: www
值: cname.vercel-dns.com
```

推荐同时添加两个，然后在 Vercel 设置重定向。

### 4.4 等待生效

DNS 生效需要 1-10 分钟（最多48小时，但通常很快）。生效后 Vercel 会自动申请 SSL 证书，你的网站就有了 HTTPS。

## 五、生成订阅码

部署完成后，生成用于售卖的订阅码：

```bash
# 在本地执行（使用服务器上的 ADMIN_API_KEY）
curl -X POST https://你的域名.com/api/admin/generate-codes \
  -H "X-API-Key: 你的ADMIN_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"count": 50, "tier": "pro", "months": 1}'
```

返回的订阅码在 Gumroad 上按个售卖，用户购买后在网站"账户"页面输入即可激活。

或者本地执行：
```bash
cd C:\Users\LC\projects\defi-saas
node scripts/generate-codes.js 50 pro 1
```

## 八、常见问题

**Q: 部署后访问显示 404？**
A: 检查 vercel.json 是否在项目根目录。

**Q: 分析功能不工作？**
A: 检查 ETHERSCAN_API_KEY 环境变量是否正确设置。

**Q: 免费额度够用吗？**
A: Vercel 免费版 100 万次请求/月，按每天 1000 次查询算，约 3 万次/月，远远低于限制。等月收入到 $500+ 再考虑升级。

**Q: 如何防止滥用？**
A: 代码已内置速率限制（每分钟30次）、免费版日配额（5次/天）、API Key 认证。如需更严格，可在 Vercel 设置 WAF 规则。

**Q: 如何迁移到自己的服务器？**
A: 任何支持 Node.js 的服务器都可以运行。复制代码，`npm install && npm start`，用 PM2 守护进程即可。
